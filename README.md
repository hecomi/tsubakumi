tsubakumi
=============

これは何？
--------------
Julius と iRemocon を用いた音声認識家電コントロールシステムです。
ブラウザから家電の登録/テストや、マクロ等を作成することが出来、OpenJTalk による TTS で返事を返してくれます。

動作環境
--------------
Mac OS X と Ubuntu 10.04 にて確認しています。MongoDB の動作が必要です。

使い方
--------------
```javascript
$ node app
// ブラウザから localhost:10080 にアクセス
```
